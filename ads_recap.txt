RÉCAPITULATIF DES ESPACES PUBLICITAIRES
=======================================

1. BANNIÈRE D'EN-TÊTE (Header Banner)
   - Code        : header-banner
   - Dimensions  : 1440px (Largeur) x 100px (Hauteur)
   - Ratio       : 14.40 : 1
   - Image recommandée : 1440 x 100 pixels (ou multiple : 2880 x 200)

2. BARRE LATÉRALE (Sidebar Top & Bottom)
   - Codes       : sidebar-top, sidebar-bottom
   - Dimensions  : 300px (Largeur) x 250px (Hauteur)
   - Ratio       : 1.20 : 1
   - Image recommandée : 300 x 250 pixels (ou multiple : 600 x 500)

3. PIED DE PAGE (Footer Pub 1 à 4)
   Les espaces du footer sont flexibles et peuvent être fusionnés selon le niveau de qualité de l'annonce.

   A. NIVEAU STANDARD (1 Bloc)
      - Utilisation : Occupation par défaut d'un seul emplacement.
      - Dimensions  : 220px (Largeur) x 210px (Hauteur)
      - Ratio       : 1.05 : 1
      - Image recommandée : 220 x 210 pixels

   B. NIVEAU MÉDIUM (2 Blocs fusionnés)
      - Utilisation : Fusionne l'emplacement actuel avec celui de droite.
      - Calcul      : (220px * 2) + 10px (gouttière)
      - Dimensions  : 450px (Largeur) x 210px (Hauteur)
      - Ratio       : 2.14 : 1
      - Image recommandée : 450 x 210 pixels

   C. NIVEAU PRÉMIUM (4 Blocs fusionnés)
      - Utilisation : Occupe toute la largeur du conteneur footer (les 4 emplacements).
      - Calcul      : (220px * 4) + (10px * 3 gouttières)
      - Dimensions  : 910px (Largeur) x 210px (Hauteur)
      - Ratio       : 4.33 : 1
      - Image recommandée : 910 x 210 pixels

---------------------------------------
Note : Les dimensions du footer incluent une gouttière (gap) de 10px entre les blocs.
Pour une netteté optimale sur les écrans haute densité (Retina), vous pouvez utiliser des images ayant le double des dimensions recommandées tout en conservant le même ratio.
